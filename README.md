# Customer Analysis using SQL

## Project Overview
This project analyzes customer data from an online store using SQL.
The analysis is performed in Google Colab using DuckDB.

## Dataset
- Customers dataset (99k+ records)
- Columns: customer_id, customer_city, customer_state, customer_zip_code_prefix

## Tools Used
- SQL
- DuckDB
- Google Colab
- Python (for visualization)

## Key Analysis
- Total customers
- Customers per state
- Top 10 cities by customers
- Customer distribution percentage by state
- Zip code analysis

## Outcome
This project demonstrates SQL skills such as SELECT, GROUP BY, ORDER BY,
aggregate functions, and basic data visualization.
